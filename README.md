# AnimeZone – Frontend Anime Website

AnimeZone is a multi-page frontend web project created to present anime content through an interactive and visually organized user experience.

The project was developed as part of academic web design requirements, with a focus on applying practical frontend concepts and building a clean, user-friendly interface.

---

## Features

- Multi-page website structure
- Home page with modal interaction
- Anime gallery with category filtering
- Hover-based content interactions
- Featured wallpapers slider
- Contact form with input validation
- Login and registration pages
- Anime ratings table
- Footer with navigation and social media icons
- Responsive design elements

---

## Pages

- Home
- Gallery
- Contact Us
- Login
- Register

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery
- Font Awesome

---

## Project Purpose

This project was built to practice frontend web development and implement academic requirements in a practical way.

It helped me improve my understanding of:
- page layout structure
- responsive design
- UI interaction
- form handling and validation
- filtering and modal implementation
